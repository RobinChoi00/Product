The Titan Vending V03 is a vending massage chair with an integrated payment cabinet, designed for premium massage experiences in high-traffic environments. Its space-saving front sliding design enables easy wall placement, while the enclosed capsule structure enhances privacy and comfort. Built for unattended operation, the V03 combines durable materials, intelligent seat detection, and flexible payment options to maximize efficiency and user satisfaction.

HOW TO USE THE V03
1. SELECT THE CHAIR
Please select the massage chair by pressing button.
2. CHOOSE A PAYMENT METHOD
Complete payment using the available options (coin or card).
3. ENJOY THE SESSION
Once payment is confirmed, the selected massage chair will automatically activate.
SESSION ENDS AUTOMATICALLY:
Remaining time is displayed on the control panel. The massage will stop automatically when the session time expires.
MINIMUM DOORWAY
Measuring Guide
  • Calculate the doorway width by subtracting 1 inch from the smallest inner width of the door frame. (1/2 inch from each side)
  • Please use a standard measuring tape to measure the doorway.
  • If the door cannot be fully opened, open it as much as possible and measure the width from the edge of the door hinge.
  • If there is a rounded curve on the wall, calculate the inner width based on the most protruding point of the wall.
